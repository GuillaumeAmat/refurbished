# Refurbished!

Unofficial Back Market themed web video game.

## Controls

| Action    | Gamepad | Keyboard player #1 | Keyboard player #2 |
| --------- | ------- | ------------------ | ------------------ |
| Dash      | `B`     | `AltLeft`          | `AltRight`         |
| Grab/Drop | `A`     | `Space`            | `Enter`            |
| Action    | `X`     | `ShiftLeft`        | `ShiftRight`       |
| Mute      | —       | `M`                | `M`                |

## Order System

Overcooked-inspired timed order system with tips, combos, penalties, and star ratings.

| Parameter                | Value                 |
| ------------------------ | --------------------- |
| Base points per delivery | 20                    |
| Order duration           | 60s                   |
| Green zone (tip +8)      | 0–40% elapsed         |
| Yellow zone (tip +4)     | 40–70% elapsed        |
| Red zone (tip +0)        | 70–100% elapsed       |
| Expired penalty          | -10 pts               |
| Combo multiplier cap     | x4                    |
| Combo applies to         | Base + tip            |
| Max simultaneous orders  | 4                     |
| Spawn interval           | 15–20s (randomized)   |
| First order delay        | 3s                    |
| Min active orders        | 1                     |
| Star thresholds          | 1★=80, 2★=200, 3★=400 |
